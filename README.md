# Hubitat Hydrawise Irrigation Controller Integration

![Hydrawise Irrigation Controller](https://raw.githubusercontent.com/KurtSanders/Hubitat-Xbar/main/Images/Hubitat-Xbar-Macbook-Pro.png)


Add "Hunter Hydrawise System", "Hunter Hydrawise Controller", and "Hubitat Hydrawise Zone" as Driver code in the Hubitat Hub 'Drivers Code' section.

Create a virtual device instance of the Hunter Hydrawise System.

Open the Hunter Hydrawise System device configuration page.  Enter your api_key and click Save.

Refresh the page, and you should see child instances of Hunter Hydrawise Controller for each controller in your system.

Click to open one of the controller devices, and you should see child devices on it for all zones configured for that controller
